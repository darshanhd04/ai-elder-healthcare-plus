Frontend Setup (Expo)
---------------------

1. Install Node.js (LTS) and npm.
2. cd frontend
3. npm install
4. Configure Expo:
   - Add your SUPABASE_URL and SUPABASE_ANON_KEY to expo.extra in app.json or eas.json, or use environment variables.
5. Start:
   npx expo start
6. Use a physical device (Expo Go) to test push notifications.
